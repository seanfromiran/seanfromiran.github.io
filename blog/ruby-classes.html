When we talk about "instance of a class" what we mean is that we have defined a new object which will now have all the properties of the Die class.

Now, we naturally arrive at the question of instance variables and methods. Instance variables are preceded by the "@" symbol and we typically define them with the arguments passed for a new instance of a class. We like instance variables because they can be used between the methods in our class. It is important to note that instance variables are defined specifically for every instance of a class. As far as instance methods, these are methods that we call on instances of our class which allow us to manipulate or display our instance variables in some way.
